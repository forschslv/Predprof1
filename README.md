# Командный кейс №2 "Управление столовой" (Predprof1-1)

## Краткое описание проекта
Веб-приложение для автоматизации работы столовой: оформление заказов, управление меню, администрирование пользователей и заказов. Реализовано на Python (FastAPI) с фронтендом на HTML/CSS/JS и хранением данных в SQLite.

## Инструкция по установке и развертыванию
1. Клонируйте репозиторий:

 git clone https://github.com/forschslv/Predprof1.git
 
2. Перейдите в папку проекта:

 cd Predprof1/NewAtt
 
3. Установите зависимости:

 pip install -r requirements.txt
 
4. Запустите сервер:
    uvicorn main:app --reload
 
5. Откройте в браузере http://127.0.0.1:8000
